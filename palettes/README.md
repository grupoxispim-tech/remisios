# Paletas Remisio

Arquivos gerados a partir de `logo16.jpg` com paletas e um preview HTML.

- GPL: arquivos de paleta importáveis no GIMP (`.gpl`).
- `remisio.css`: variáveis CSS com as cores principais.
- `preview.html`: preview visual das paletas (abra no navegador).
 - `swatches-*.svg`: arquivos vetoriais com blocos das paletas (abra no navegador ou importe no Figma).

Exportar para PNG
- Abra o SVG no navegador e salve como PNG usando ferramentas de captura, ou importe o SVG no Figma/Illustrator e exporte como PNG nos tamanhos desejados.


Importar `.gpl` no GIMP: coloque o arquivo em `~/.gimp-2.8/palettes/` ou use menu Palettes > Import.
