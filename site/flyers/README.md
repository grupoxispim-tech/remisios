# Flyers — Remisio's Pizza

Arquivos SVG prontos para divulgação. Você pode abrir os SVGs em um navegador, editor vetorial (Inkscape, Illustrator) ou importar no Figma.

Arquivos:
- `flyer-social-1080.svg` — banner quadrado 1080x1080 (Instagram feed)
- `flyer-story-1080x1920.svg` — formato Story/Reel 1080x1920
- `flyer-a4.svg` — A4 (pronto para exportar em PDF)

Exportar para PNG (Inkscape):
```bash
inkscape flyer-social-1080.svg --export-type=png --export-filename=flyer-social-1080.png
inkscape flyer-story-1080x1920.svg --export-type=png --export-filename=flyer-story-1080x1920.png
inkscape flyer-a4.svg --export-type=pdf --export-filename=flyer-a4.pdf
```

Se preferir usar o navegador: abra o SVG, faça captura em tela ou importe no Figma e exporte nos formatos desejados.

Antes de publicar, atualize os links do WhatsApp (`wa.me`) no HTML com o número correto.
